## Projects

- **Safe Step** — Wearable fall detection system for elderly care. 96% sensitivity, <1s BLE alert latency. Built on BBC micro:bit V2 with custom C++ algorithm.
- **Polyphonic Piano & Sequencer** — 6-note analog synthesizer using 556 timers, LM741 summing amplifier, and LM386 audio output. No microcontrollers.
- **Pill Crusher Tool** — Ergonomic, accessible pill crusher with lever-based mechanical advantage, Braille labeling, and slide-out pill tray.
- **Mousetrap Car** — Energy-optimized vehicle built through iterative mechanical analysis of wheel geometry, lever arm, and friction reduction.
- **KidWind Wind Turbine** — Custom wind turbine (CAD, laser-cut blades, 3D-printed gearbox, PVC base) that won 1st place nationally at the NSBE KidWind Open Division Championship. 0.52W output at 83.3% electrical efficiency.
