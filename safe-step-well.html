Safe Step

Intelligent fall detection system for elderly care

Overview

Executive summary.

Safe Step is an intelligent, wearable fall detection system providing real-time alerts and reduced emergency response time for elderly adults. Combining sensor fusion, custom C++ algorithms, and privacy-first local processing, it delivers professional-grade fall detection at a fraction of the cost of existing solutions.

The gap in elderly care.

Algorithm

3-stage detection pipeline.

Stage 1
Freefall Detection

Detects rapid acceleration drop indicating loss of ground contact. Triggers when G-force drops below 0.5G — threshold adjustable per user profile.

Stage 2
Impact Confirmation

Validates fall by detecting a high-G spike (>2.5G) following the freefall phase. Eliminates false positives from sudden sitting or jumping, which lack the proper deceleration signature.

Stage 3
Orientation Validation

Uses gyroscope data to confirm horizontal posture post-impact. Validates position for 500–1000ms before triggering caregiver alert.
