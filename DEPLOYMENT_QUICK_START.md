## ⚡ 5-Minute Setup

### 1. Create GitHub Account (if needed)
→ Go to [github.com](https://github.com) and sign up

### 2. Create Repository
→ Click **+** → **New Repository**  
→ Name it: **gabrielkinney.github.io** (exact name required!)  
→ Choose **Public**  
→ Click **Create**  

### 3. Upload Files
→ Click **Add file** → **Upload files**  
→ Select all .html files from your computer  
→ Click **Commit changes**  

### 4. Enable GitHub Pages
→ Click **Settings**  
→ Scroll to **Pages**  
→ Source: select **main** branch  
→ Click **Save**  

### 5. View Your Portfolio
→ Wait 2-3 minutes  
→ Visit: **https://gabrielkinney.github.io** ✅

## 🆘 Common Issues

**Portfolio not showing?**
- Wait 2-3 minutes for GitHub Pages to build
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

**Links broken?**
- Check file names are exact case-sensitive
- Ensure all HTML files uploaded to same folder

**Need to update contact info?**
- Edit index.html line ~350 (contact section)
- Update email and LinkedIn URLs
- Re-upload file
